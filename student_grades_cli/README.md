# Student Grades CLI (Mini Project)

A simple command-line program that:

- Collects student names and two exam grades using `input()`
- Combines the data with `zip()`
- Calculates averages with `map()`
- Filters approved students with `filter()`
- Generates approval messages with `map()`

## How to run

```bash
python student_grades_cli.py
```

## Concepts practiced

- `input()`, `for` + `range`
- Lists
- `zip()`
- `map()` + `lambda`
- `filter()` + `lambda`
- Basic formatting with f-strings
