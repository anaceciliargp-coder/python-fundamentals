# Python Fundamentals

This repository contains my Python fundamentals practice, focused on building a solid foundation in programming logic, data structures, and problem-solving.

The exercises and mini-projects were developed during my studies and reflect hands-on practice of core Python concepts commonly expected for junior-level roles. The focus is on clarity, consistency, and gradual progression in complexity.
