### **lead-prequalification-cli/**
A small **command-line mini project** simulating a **lead pre-qualification flow**, built using only Python fundamentals.

**Concepts applied:**
- User input handling  
- Data validation  
- Conditional logic  
- Functions and return values  
- Simple business rules  
- Clean and readable control flow  

**What the program does:**
- Collects basic lead information (name, age, monthly income)  
- Validates user input  
- Allows the user to confirm or re-enter their information  
- Classifies the lead as:
  - `Invalid`
  - `Basic`
  - `Qualified`
- Generates an appropriate message based on the classification  

This project was intentionally built **without frameworks, external APIs, or advanced techniques**, focusing purely on **logic and fundamentals**, making it suitable as an early portfolio project.
